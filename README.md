# Startover Extra

Offline backup orchestrator for [Startover](https://howtoplay.mystrikingly.com/) websites.

Built as a Firefox extension that is able to backup Strikingly sites as single-file HTML documents.

The extension generates a sidebar and loads the sites, extracts the URLs and is able to generate a single-file HTML document of the page with all the images and stylesheets embedded. The scripts
are removed. The YouTube & Vimeo embeds from embedly are replaced with regular embeds. The JS-dependant navigation bar is replaced with a static navigation bar. Superflous elements and branding are removed.
